# INTENT TRAVEL

**Intent Travel** is a financial intent engine designed to solve the "Group Chat Death Spiral." It replaces passive agreement with verifiable commitment, turning travel dreams into data-driven departures.

---

## 💡 The Problem
Most group trips die in the planning phase because talk is cheap. In a typical group chat, everyone says they are "interested," but no one is committed. This lack of **tangible intent** leads to indecision, missed bookings, and travel plans that never leave the chat.

## 🚀 The Solution: Financial Intent Stacking
Vouch introduces a **Consensus Leaderboard**. Instead of voting with emojis, group members "vouch" for destinations with a theoretical financial stake. 

* **Quantifiable Interest:** We aggregate total "vouched" amounts to show which destination has the most actual backing.
* **Real-Time Visualization:** Using Chart.js, we provide a dynamic "Heat Map" of group consensus.
* **Frictionless Commitment:** By visualizing the financial weight of an opinion, groups can move from debate to booking with confidence.

---

## 🛠️ Technical Stack
* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS (Custom "Soft-UI" / Bubbly Aesthetic)
* **Data Visualization:** Chart.js / React-Chartjs-2
* **Icons:** Lucide-React
* **Architecture:** Component-driven design for rapid scaling and modularity.

---

## 🧠 Key Features
* **The Engine:** A custom aggregator that sorts and ranks destinations based on total financial weight.
* **Dynamic Analytics:** A responsive bar chart that updates in real-time as new "stakes" are added.
* **Group Sync:** Unique group ID generation to simulate a shared workspace environment.
* **Soft-UI Design:** A "no-edge" aesthetic built with custom Tailwind configurations and the Fredoka/Quicksand font families for a modern, friendly user experience.
